Структура директорий:
	
<дата-записи> \
    
		<название-аудитории> \
        
			<название-аудитории [ <дата-записи>_<время-начала-записи> ].mpg>



Формат файлов:

	video: h264 - mpeg-4 avc
    
	audio: mpga - mpeg audio layer 3 (mp3)
    
	контейнер: mpeg-ts
    


Для поддержки mpga(mp3) звука в стандартном Windows Media Player 
нужно установить дополнительный кодек из пакета ffdshow_codec.exe. 

Либо воспользоваться альтернативными видеоплеерами, например VLC.
 
Необходимые файлы приложены в директории redistr.
